# Claude Code Configuration

This directory contains configuration and context for Claude Code to work effectively with the Bloginator project.

## Files

- `project-context.md` - Project overview and architecture
- `coding-standards.md` - Coding conventions and standards
- `common-tasks.md` - Frequently performed development tasks

## Usage

Claude Code automatically reads these files to understand:
- Project structure and conventions
- Quality requirements and testing standards
- Common development workflows

## Updating

Keep these files synchronized with:
- `docs/CLAUDE_GUIDELINES.md` - Overall project guidelines
- `docs/DESIGN-SPEC-001-Implementation-Plan.md` - Implementation details
- `docs/TESTING-SPEC-003-Quality-Assurance.md` - Quality standards
